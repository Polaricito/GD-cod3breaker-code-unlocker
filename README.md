# GD-cod3breaker-code-unlocker

A Python script to solve the Geometry Dash puzzle: cod3breaker

Instructions:
1. Clone the repository (also you can download and execute gd.py)
1. Record the numbers that appear (you can also write them down somewhere)
2. Enter the numbers in the order they appear
3. Copy the resulting code and paste it into the Vault of Secrets
4. Done! Enjoy your new icon
