# GD-cod3breaker-code-unlocker

[Español](https://github.com/Polaricito/GD-cod3breaker-code-unlocker/blob/main/READMEes.md)

A Python script to solve the Geometry Dash puzzle: cod3breaker

Instructions:
1. Download the latest release for your platform
2. Unzip it and execute it
3. Record the numbers that appear (you can also write them down somewhere)
4. Enter the numbers in the order they appear
5. Copy the resulting code and paste it into the Vault of Secrets
6. Done! Enjoy your new icon
